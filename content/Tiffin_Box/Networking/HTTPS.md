#basic #networking 

HTTPS stands for *Hyper Text Transfer Protocol Secure*. HTTPS is the modified version of [[HTTP]]. The only difference is encryption. Every data moves with encryption.

[[HTTP]] send everything in plain text, so anyone between browser and the server can read those data. HTTPS works in two phases — first a [[TLS handshake]] to establish a secure tunnel, then normal HTTP flows through that tunnel, completely encrypted.

### Why we need HTTPS??
- Secure communication
- User privacy protection
- Protection from hackers
- Safe online transaction
- Improve website credibility
- Better SEO and trust
